Recent Projects

Below are three recent projects I have been able to be apart of or completed myself

Podcast Planner

Podcast Planner is a web app that allows the user to specify the length of time they wish to listen to one or more podcasts then filters the search results accordingly. The returned searches can be added to a personal playlist that stores the users information in the browser using Firebase. This was a project that was built with three other developers using React and Firebase.

Space Out

Space Out is a web app that allows the to track where the International Space Station is currently, as well as calculate how far the user is from the Space Station at any given time. This was a project that was created with one other developer using paired programming, this is a API-powered application written with jQuery.

Cook Book

Cook Book is a web app that generates random recipes for the user using a JSON database to store the selected recipes.This was a project that was a solo project, this is a Firebase powered application written using react.
